# Таймер + заметки
Многофункциональное десктопное приложение на Python с графическим интерфейсом, объединяющее таймер и систему управления задачами с сохранением данных.

# Возможности
1. Быстрый выбор времени с предустановленными интервалами в минутах (1, 5, 10, 15, 30, 45, 60, 90, 120)
2. Таймер продолжает отсчёт времени, даже после закрытия программы
3. Есть возможность установить время в ручную(часы, минуты)
4. Имеются визуальные индикаторы в виде цветов(Зелёный - таймер запущен, Жёлтый - осталось менее 10 минут и Красный - осталось менее 5 минут)
5. Также при окончании времени программа сообщит об этом, подаст звуковой сигнал.

# Управление задачами
Есть возможность создать задачи с указанием:
1. Название задачи
2. Выставление даты и времени окончания задачи(есть встроенный календарь)
3. Выбор сложности задачи (Не Важная, Важная, Очень Важная, Совсем Важная)
4. Возможность выбирать прогресс определённой задачи с помощью ползунка
5. Реализовано добавление подзадач для каждой задачи
6. Имеется выбор категории задач(Активные, Заверщённые и ПРосроченные задачи). Просроченные задачи автоматически добавляются в соответствующий раздел

# Сохранение данных
1. Автомтаическое сохранение заметок в JSON файл с возможностью просмотра
2. Сохранение состояния таймера между сессиями
3. Данные востанавливаются при запуске программы
4. Сохранение данных при случайном закрытии программы

# Установка и требования
### Требования
1. Python 3.8 и выше
2. Операционная система только на Windows
### Установка
1. Клонирование репозитория
git clone https://github.com/your-username/Timer1.git
cd Timer1
2. Установкиа зависимостей(Не требуестся используются стандартные библиотеки Python)
Список библиотек для проекта:

# Использование
### Вкладка Таймер
1. Выберите время с помощью кнопок быстрого выбора или введите вручную
2. Нажмите "Старт" для запуска таймера
3. Используйте кнопку "Пауза" для временной остановки
4. С помощь кнопки "Сброс" таймер сбрасывает время полностью

### Вкладка Заметки
1. Введите название задачи в поле "Задача"
2. Выберите дату и время окончания:
Введите вручную в формате ДД.ММ.ГГГГ ЧЧ:ММ или Нажмите кнопку календарь для выбора через календарь
3. Выберите уровень сложности из выпадающего списка
4. Установите начальный прогресс (опционально)
5.Нажмите "Добавить заметку"

### Дополнительные функции:
Подзадачи: выберите задачу → введите текст подзадачи → нажмите "Добавить подзадачу"
Конвертация в часы: выберите задачу → нажмите "Конвертировать в часы" для установки времени в таймер
Отметка выполнения: выберите задачу → нажмите "Отметить как выполненную"
Удаление: выберите задачу → нажмите "Удалить заметку"

### Вкладки "Завершённые" и "Просроченные"
1. Просмотр истории выполненных и просроченных задач
2. Возможность восстановления задач в активные
3. Удаление записей из истории

# Технические детали
Сохранение данных
1. Файл таймера (timer_state.json): содержит время окончания и статус работы таймера

2. Файл заметок (notes_data.json): содержит все заметки в формате JSON с полями:
2.1 active_notes - активные задачи
2.2 completed_notes - завершённые задачи
2.3 failed_notes - просроченные задачи

# Ограничения Таймера
1. Таймер работает только в фоновом режиме при закрытии с подтверждением
2. Звуковые сигналы работают только в Windows (используется winsound)
3. Для работы календаря требуется корректный системный шрифт Arial
